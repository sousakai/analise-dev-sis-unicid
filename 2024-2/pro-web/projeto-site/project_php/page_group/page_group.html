Estruturas PHP Utilizadas
Home Login

Na página home login, foram utilizados os comandos:

  • session_start(): para iniciar a sessão;
  • if (isset($_SESSION['erro'])): para exibir o alerta em caso de erro de login;
  • unset($_SESSION['erro']): para garantir que a mensagem de erro possa ser eliminada.

Login

Na página login, foram utilizados os comandos:

  • session_start(): para iniciar a sessão;
  • $usuario_valido e $senha_valida: para definir o usuário;
  • if ($_SERVER["REQUEST_METHOD"] == "POST"): para verificar se os dados foram enviados via método POST;
  • if ($usuario === $usuario_valido && $senha === $senha_valida): para verificar se a senha e o usuário estão corretos;
  • $_SESSION['username'] = $usuario: para armazenar o nome do usuário;
  • header("Location: ../reserva_ingress/reserva_ingress.php"): para redirecionar o usuário já logado.
  • exit(): para interromper o script.

Logout

Na página logout, foram utilizados os comandos:

  • session_start(): para iniciar a sessão;
  • session_destroy(): para destruir todas as variáveis de sessão;
  • header("Location:homeLogin.php"): para redirecionar para a página de login.
